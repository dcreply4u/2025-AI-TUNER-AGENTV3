# TelemetryIQ Website Assets Brief

**For Designer/Developer**  
**Date**: 2025  
**Status**: Ready for Production

---

## Overview

This document specifies all logo variations and website assets needed for the TelemetryIQ website. The approved logo design is provided as reference, and all assets should be created based on this design.

---

## Approved Logo Reference

The approved logo features:
- Circular tachometer/gauge emblem with checkered flag outer ring
- "IQ" center element in glowing electric blue with neural network patterns
- "TELEMETRYIQ" wordmark below in bold electric blue
- Red needle pointing to redline zone (numbers 4-5-6)
- Electric blue (#00E0FF) and racing red (#FF3B3B) color scheme
- Dark background (#0D1117) with electric blue glow effects

**Note**: There should be NO tagline, NO subtitle, and NO additional text below "TELEMETRYIQ" wordmark.

---

## Required Logo Variations

### 1. Full Color Logo (Primary)
- **Format**: SVG, PNG (transparent)
- **Sizes**: 
  - SVG (vector, scalable)
  - PNG 1024x1024px
  - PNG 2048x2048px
- **Use**: Main website logo, hero sections, headers
- **Colors**: Full electric blue, racing red, dark background

### 2. White Logo (Dark Backgrounds)
- **Format**: SVG, PNG (transparent)
- **Sizes**: Same as above
- **Use**: Dark backgrounds, night mode
- **Colors**: White/light gray with electric blue accents

### 3. Black Logo (Light Backgrounds)
- **Format**: SVG, PNG (transparent)
- **Sizes**: Same as above
- **Use**: Light backgrounds, print materials
- **Colors**: Black/dark gray

### 4. Icon Only (Favicon & App Icons)
- **Format**: SVG, PNG, ICO
- **Sizes Required**:
  - 16x16px (favicon)
  - 32x32px (favicon)
  - 48x48px (favicon)
  - 64x64px (app icon)
  - 128x128px (app icon)
  - 256x256px (app icon)
  - 512x512px (app icon)
  - 1024x1024px (app icon)
- **Use**: Browser favicon, app icons, small spaces
- **Note**: Simplify for smallest sizes (16x16, 32x32) - may need minimal detail version

### 5. Wordmark Only (Horizontal)
- **Format**: SVG, PNG (transparent)
- **Sizes**: 
  - SVG (vector, scalable)
  - PNG 1024px width (height proportional)
- **Use**: Headers, navigation bars, horizontal spaces
- **Text**: "TELEMETRYIQ" only (no gauge emblem)

## Website-Specific Assets

### Header/Navigation Logo
- **Format**: SVG (preferred) or PNG
- **Size**: 200px height (width proportional)
- **Variations**: 
  - Full color (default)
  - White version (for dark header)
  - Black version (for light header)
- **Use**: Website header, navigation bar

### Hero Section Logo
- **Format**: SVG or PNG
- **Size**: 400-600px height (width proportional)
- **Variations**: Full color with glow effects
- **Use**: Hero section, landing page main logo

### Footer Logo
- **Format**: SVG or PNG
- **Size**: 150px height (width proportional)
- **Variations**: 
  - White version (for dark footer)
  - Black version (for light footer)
- **Use**: Website footer

### Favicon Set
- **Format**: ICO, PNG, SVG
- **Sizes**: 
  - 16x16px
  - 32x32px
  - 48x48px
- **Variations**: Icon-only, simplified for small size
- **Use**: Browser tab icon

### Apple Touch Icons
- **Format**: PNG
- **Sizes**:
  - 180x180px (iPhone)
  - 152x152px (iPad)
  - 167x167px (iPad Pro)
- **Use**: iOS home screen icons

### Android Icons
- **Format**: PNG
- **Sizes**:
  - 192x192px
  - 512x512px
- **Use**: Android home screen icons

### Social Media Assets

#### Profile Pictures
- **Format**: PNG (square)
- **Size**: 400x400px
- **Use**: Social media profile images
- **Variation**: Icon-only version

#### Cover Images
- **Format**: PNG or JPG
- **Sizes**:
  - Twitter: 1500x500px
  - Facebook: 820x312px
  - LinkedIn: 1584x396px
  - YouTube: 2560x1440px
- **Use**: Social media cover/banner images
- **Note**: Include logo and brand elements, but not just logo alone

#### Open Graph Image
- **Format**: PNG or JPG
- **Size**: 1200x630px
- **Use**: Social media link previews
- **Content**: Logo + brand name, professional design

## Technical Requirements

### SVG Files
- Clean, optimized code
- Properly named layers/groups
- Scalable without quality loss
- Compatible with modern browsers

### PNG Files
- Transparent background (where applicable)
- High resolution (2x for retina displays)
- Optimized file size
- No compression artifacts

### ICO Files
- Multiple sizes embedded (16x16, 32x32, 48x48)
- Proper format for browser compatibility

## Design Notes

### Modifications Allowed
- ✅ Adjusting sizes and proportions
- ✅ Creating simplified versions for small sizes
- ✅ Adjusting glow effects for different backgrounds
- ✅ Creating variations for different use cases
- ✅ Optimizing for web performance

### Modifications NOT Allowed
- ❌ Changing core design elements (gauge, "IQ", wordmark)
- ❌ Adding taglines or subtitles
- ❌ Changing approved colors (except for variations)
- ❌ Removing essential elements
- ❌ Changing typography style significantly
