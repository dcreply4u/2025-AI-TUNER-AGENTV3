## AI Tools to Use

### Image Generation
- **DALL-E 3** (OpenAI) - Best for logos, product images
- **Midjourney** - Best for artistic, high-quality images
- **Stable Diffusion** - Open source, customizable
- **Canva AI** - Quick social media graphics

### Video Generation
- **Runway ML** - AI video generation
- **Pika Labs** - Text-to-video
- **Synthesia** - AI avatars for explainer videos
- **Luma AI** - 3D video generation

### Content Writing
- **ChatGPT** (GPT-4) - Blog posts, copy
- **Claude** (Anthropic) - Long-form content
- **Jasper AI** - Marketing copy
- **Copy.ai** - Quick content generation

## Usage Instructions

### For Images
1. Copy prompt from above
2. Paste into DALL-E, Midjourney, or Stable Diffusion
3. Adjust parameters (resolution, style)
4. Generate multiple variations
5. Select best results
6. Edit if needed (remove backgrounds, adjust colors)

### For Videos
1. Use video generation prompts
2. Generate with Runway ML or Pika Labs
3. Edit with video editor (Premiere, Final Cut)
4. Add voiceover (ElevenLabs, Murf AI)
5. Add music (Epidemic Sound, Artlist)
6. Export in multiple formats

### For Blog Posts
1. Use blog post prompts
2. Generate with ChatGPT or Claude
3. Edit for brand voice
4. Add images (from image prompts)
5. Optimize for SEO
6. Publish on website/blog
