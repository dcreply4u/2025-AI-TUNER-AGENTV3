## Logo Description

The approved TelemetryIQ logo successfully combines AI intelligence with racing performance through a sophisticated tachometer/gauge design.

### Core Elements

1. **Central Gauge/Tachometer**
   - Dark metallic circular emblem
   - Checkered flag pattern on outer ring (racing symbol)
   - Concentric rings with glowing electric blue dots (RPM marks)
   - Racing red arc segment (upper right) with numbers 4-5-6 (redline zone)
   - Electric blue segments with numbers 8-9-10
   - Red needle/pointer from center pointing to redline zone

2. **"IQ" Center Element**
   - Prominently displayed in glowing electric blue (#00E0FF)
   - Stylized with internal circuit/neural network patterns
   - Forms cohesive unit (NOT separate "10" and "Q")
   - 3D depth effect

3. **AI Symbolism**
   - Neural network patterns integrated into "IQ" letters
   - Circuit traces forming AI network patterns
   - Glowing blue circuit patterns in background
   - Interconnected nodes suggesting machine learning

4. **Wordmark**
   - "TELEMETRYIQ" below emblem
   - Bold, glowing electric blue (#00E0FF)
   - 3D depth effect
   - Modern sans-serif typography
   - Single wordmark line only

5. **Color Palette**
   - **Primary**: Electric blue (#00E0FF)
   - **Accent**: Racing red (#FF3B3B)
   - **Background**: Dark (#0D1117)
   - **Effects**: Electric blue glow, subtle red accents

6. **Visual Effects**
   - Electric blue glow emanating from elements
   - 3D isometric depth
   - Subtle circuit patterns in background
   - Premium metallic appearance

## Usage Guidelines

### ✅ DO's

- Use full color version on dark backgrounds
- Use white/light version on dark backgrounds
- Use black/dark version on light backgrounds
- Maintain minimum clear space (equal to height of "I" in "IQ")
- Scale proportionally (never stretch or distort)
- Use high-resolution files for print
- Use SVG for web when possible

### ❌ DON'Ts

- Never add taglines or subtitles below wordmark
- Never modify colors (use approved palette only)
- Never rotate or tilt the logo
- Never place on busy backgrounds that reduce visibility
- Never use at sizes smaller than 16x16px
- Never add effects (shadows, glows) beyond approved design
- Never separate "IQ" from "Telemetry"
- Never use incorrect spelling ("TELEMETRYQ", "TELEMETRY", etc.)
